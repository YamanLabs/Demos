# Velocity Forge - High-Performance AI Interface

Velocity Forge is a hyper-futuristic UI design showcase for deep tech, AI, and high-performance infrastructure companies. It demonstrates sensory/machine interface aesthetics using neural network topologies.

## Core Visuals
- **Neural Network Topology**: Interactive node-and-link network with pulse simulations.
- **Hardware Cluster Matrix**: Real-time thermal and load visualization for high-performance computing clusters.
- **Precision Gauges**: Custom SVG-based monitoring equipment UI.
- **Synthesis Stream**: Command-line style log flow with glitch-text effects.

## Tech Stack
- Framework: [Next.js](https://nextjs.org/)
- Styling: [Tailwind CSS 4](https://tailwindcss.com/)
- Animation: [Framer Motion](https://www.framer.com/motion/)
- Icons: [Lucide React](https://lucide.dev/)
