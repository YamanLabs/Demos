# Lexington Oracle - Editorial Intelligence Platform

Lexington Oracle is a typographic excellence showcase for high-volume content platforms. It blends 1920s editorial aesthetics with modern minimalist UX/UI architecture.

## Design Innovations
- **Zen Reading Mode**: Paper-to-Digital transition for focused content consumption.
- **Semantic X-Ray**: Interactive insight layer that scans content for complex terms and provides AI-driven definitions.
- **Oracle Search Engine**: An expanded, dense search interface with quantum data stream animations.
- **Generative Data Globe**: Decorative SVG component representing global intelligence flow.

## Tech Stack
- Framework: [Next.js](https://nextjs.org/)
- Styling: [Tailwind CSS 4](https://tailwindcss.com/)
- Animation: [Framer Motion](https://www.framer.com/motion/)
- Icons: [Lucide React](https://lucide.dev/)
