# Aura Zenith - Interactive Visual Engine

Aura Zenith is a high-performance visual interaction showcase built with Next.js, Tailwind CSS, and Framer Motion. It demonstrates advanced light refraction, mirroring, and tactile feedback effects.

## Features
- **Refraction Layer**: Real-time glass morphism with light bending.
- **Mirroring System**: Dynamic reflection tracking based on mouse/touch position.
- **Tactile Feedback**: Interactive card system with deep CSS variable manipulation.
- **Scroll Intent Detection**: Smart switching between interaction and scrolling.

## Tech Stack
- Framework: [Next.js](https://nextjs.org/)
- Styling: [Tailwind CSS 4](https://tailwindcss.com/)
- Animation: [Framer Motion](https://www.framer.com/motion/)
- Icons: [Lucide React](https://lucide.dev/)
